# Company Logos

Add company logo images to this directory. Recommended specs:

- **Format**: PNG with transparent background (or JPG/SVG)
- **Size**: 200×200px minimum (square or contain ratio)
- **Naming**: lowercase, e.g., `sony.png`, `tfi.png`, `csf.png`

You can download logos from the companies' websites or use a placeholder service like:
- https://logo.clearbit.com/{domain} (e.g., `https://logo.clearbit.com/sony.com`)
- https://unavatar.io/{domain}

Or create simple text-based placeholders temporarily.
